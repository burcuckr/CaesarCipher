A Python implementation of the **Caesar Cipher**, a classic encryption technique where each letter in a message is shifted by a fixed number of positions in the alphabet.

The program allows the user to:
- Encrypt a message (`encode`)
- Decrypt a message (`decode`)
- Choose any shift number
- Restart the program as many times as they want

Non-alphabet characters (such as spaces and punctuation) are preserved.
